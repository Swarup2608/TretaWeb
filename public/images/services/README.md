# Services Images

This directory contains images for the services section and individual service detail pages.

## Main Banner
- `banner.jpg` - Main banner image for the services page (1920x500px recommended)

## Individual Service Images
All services are managed in `/public/SiteContent/services.json`

Each service uses the following image paths:
- `wealth-finance.jpg` - Image for Wealth & Finance Management service
- `risk-assessment.jpg` - Image for Risk Assessment & Mitigation service
- `talent-strategy.jpg` - Image for Talent Strategy & Development service
- `growth-planning.jpg` - Image for Growth Planning & Execution service
- `performance-optimize.jpg` - Image for Performance Optimization service
- `market-analysis.jpg` - Image for Market Analysis & Insights service

## Recommended Specifications
- Format: JPG or PNG
- Dimensions: 1200x800px for service detail images
- File size: Keep under 500KB for optimal loading
- Style: Professional, business-oriented imagery

## Upload Instructions
1. Add your images to this directory
2. Use the exact filenames listed above
3. Or update the image paths in `/public/SiteContent/services.json`
4. Manage all service content (including button text and links) through the dashboard at `/dashboard/services`
